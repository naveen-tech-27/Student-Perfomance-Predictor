Project Overview

Student Performance Predictor is an AI-powered web application that estimates a student's academic performance score (0–100) based on key personal and behavioral factors.

The system uses a Random Forest Regressor trained on student behavior data inspired by the UCI Student Performance Dataset. The model is served via a Python Flask API and displayed in a responsive, modern web interface.

🗂️

Dataset & Features

The model uses 8 carefully selected features that have the strongest influence on student outcomes:

  • Study Time — Weekly hours dedicated to studying
  • Past Failures — Number of previous class failures
  • School Absences — Days absent during the year
  • Internet Access — Home internet availability
  • Free Time — After-school free time (1–5 scale)
  • Health Status — Self-reported health (1–5 scale)
  • Family Relations — Family relationship quality (1–5 scale)
  • Paid Classes — Enrollment in extra paid tutoring
🤖

Machine Learning Model

The prediction pipeline consists of:

  1. Feature Encoding — Categorical values (Yes/No, ranges) are numerically encoded
  2. Standard Scaling — All features normalized using StandardScaler
  3. Random Forest Regressor — 200 trees, max depth 10, trained on 2000 synthetic samples
  4. Output — Score (0–100) + performance category + feature importances

Important Notes

Synthetic Data

The model was trained on synthetically generated data designed to mimic real student patterns. Results are for educational/demonstration purposes.

Not a Replacement

This tool is not a substitute for professional academic counseling. Use it as a motivational aid, not a definitive assessment.

Privacy

No student data is stored. All predictions are stateless — your inputs are processed and immediately discarded by the server.
